# ExoLang v0.1

ExoLang is a simple experimental command-based language made in Python.
Linux only.

## Features
- Print text
- Clear screen
- Show system specs
- Run htop / cmatrix (if installed)
- Command counter

## Requirements
- Python 3
- Linux
- neofetch, htop, cmatrix (optional)

## Run
```bash
python3 exolang.py
